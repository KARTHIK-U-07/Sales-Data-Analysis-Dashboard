# 📊 Sales Data Analysis & Dashboard

## 🚀 Project Overview
This project analyzes sales data to uncover key business insights and visualize performance using Power BI.

## 📁 Dataset
* Superstore Sales Dataset
* Includes sales, profit, customer, and regional data
  
## 🛠️ Tools Used
* Excel (Data Cleaning)
* Power BI (Dashboard)
  
## 📊 Dashboard Features
* Sales by Region
* Monthly Sales Trend
* Profit by Category
* Top Customers
* KPI Metrics (Total Sales, Profit)

## 🧠 Key Insights
* West region generated the highest sales
* Technology category had the highest profit
* Sales peaked during year-end months

## 📌 Conclusion
This project demonstrates data cleaning, analysis, and visualization skills required for a Data Analyst role.
